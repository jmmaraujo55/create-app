
# What's Poundcake?

A NodeJS application generator to scaffold new projects.

# Installation

```console
$ npm i -g create-app
```

# Usage

To generate a app:

```console
$ create-app [options] <name>
```

# Command Line Options

```console
# create-app --help
--force           remove directory if exists
--yarn            use yarn (default npm)
-h, --help        display help for command
```
